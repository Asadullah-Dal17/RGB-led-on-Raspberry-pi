# RGB-led-on-Raspberry-pi
> Controlling RGB🔴🟢🔵 LED on Raspberry Pi 4, Basics 


## Pins Number for RGB LED


- Red   🔴 LED PIN 26  ➡️ 2️⃣6️⃣
- Green 🟢 LED PIN 19  ➡️ 1️⃣9️⃣
- Blue  🔵 LED pin 13  ➡️ 1️⃣2️⃣
